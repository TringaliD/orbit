# 🌌 ORBIT — World Social Map (Neon Edition)

Orbit è un prototipo di social media con mappa neon globale.  
Gli utenti possono pubblicare una foto dalla loro posizione e vederla apparire come marker luminoso.

## 🚀 Funzionalità
- Mappa neon interattiva (Carto Dark Matter)
- Upload foto dal dispositivo
- Geolocalizzazione automatica
- Marker con glow e popup visivo

## ▶️ Come usarlo
1. Clona il repository
2. Apri `index.html` nel browser
3. Clicca “Pubblica Foto”
4. Carica una foto
5. Orbit la posiziona sulla tua posizione attuale
