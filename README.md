# 🌌 ORBIT — World Social Map

Orbit è un prototipo di social media basato sulla mappa del mondo.  
Gli utenti possono pubblicare una foto dalla loro posizione e vederla apparire come un marker sulla mappa.

## 🚀 Funzionalità
- Mappa globale interattiva (Leaflet.js)
- Upload foto dal dispositivo
- Geolocalizzazione automatica
- Marker con popup contenente la foto
